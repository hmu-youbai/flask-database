HOME PAGE

The home page provides details about this database, which contains ten datasets. It also outlines the lineage differentiation process of the human early embryo and highlights the corresponding markers of different cell types at various developmental stages.



SEARCH PAGE (Single Gene)

The HPPIED page offers a user-friendly search interface for gene data exploration. Users can click the search button to explore data based on gene information and choose either ENSEMBL ID or Gene Symbol for their search. The search page displays feature plots, violin plots of expression levels for the input gene, and line plots illustrating different developmental stages and cell types simultaneously.
